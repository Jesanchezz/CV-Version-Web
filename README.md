# Curriculum Vitae - Versión HTML

Este proyecto es la versión en código web de mi Curriculum Vitae. La idea principal es llevar mi CV tradicional a la web utilizando tecnologías como **HTML**, **CSS** y **JavaScript**. También, he añadido un modo oscuro para una experiencia visual más agradable y comoda.

## Objetivo
Crear una presentación profesional, interactiva y visualmente atractiva de mi experiencia, habilidades y formación, accesible desde cualquier dispositivo con navegador web.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## ¿Por qué lo hice?
Quise transformar mi currículum en una experiencia digital, mostrando mis conocimientos en desarrollo frontend y facilitando su acceso y actualización.
